# SlushAi
СлушАй (Сводка собраний/лекций/конференций) — интеллектуальная десктопное приложение для автоматической расшифровки, анализа и структурирования аудиозаписей встреч и лекций. Позволяет быстро получать текстовые транскрипты, формировать сводки, выделять ключевые моменты обсуждений.

Чтобы установить все нужные библиотеки, пропишите в вашей среде:
```
pip install -r requirements.txt
```
