# СлушАй (SlushAi)

СлушАй — интеллектуальное десктопное приложение для автоматической расшифровки, анализа и структурирования аудиозаписей встреч, лекций и конференций.  
Оно помогает экономить время, превращая «сырые» записи в удобные для чтения конспекты, сводки и списки ключевых моментов.

## О проекте

СлушАй решает типичную проблему: после встречи или пары остается длинная аудиозапись, в которой сложно быстро найти нужный фрагмент.  
Приложение автоматически преобразует аудио в текст, выделяет важные части и помогает навести порядок в информационном хаосе.  
Интерфейс спроектирован так, чтобы было удобно загружать записи, просматривать транскрипт, сводку и структуру обсуждения в одном окне.

## Основные возможности

- **Автоматическая расшифровка с ИИ:** Преобразование аудиозаписей (встречи, лекции, конференции) в точный текстовый формат с использованием локальной ИИ-модели, что обеспечивает конфиденциальность ваших данных.
- **Реферирование (Summary):** Генерация краткого содержания записи для быстрого ознакомления — вы сразу поймете суть без прослушивания часов аудио.
- **Выделение главного:** Локальный ИИ автоматически извлекает ключевые моменты: принятые решения, поставленные задачи, вопросы и важные идеи.
- **Структурирование контента:** Текст автоматически разбивается на логические блоки: темы обсуждений и временные метки для навигации.
- **Полная автономность:** Это локальное десктопное приложение. Все вычисления происходят на вашем железе после установки зависимостей — интернет для работы моделей не нужен.

## Установка и запуск

1. Установите Python 3.12.7 (актуальную стабильную версию).  
2. Клонируйте репозиторий:

```
git clone https://github.com/leriuszxc/SlushAi.git
cd SlushAi
```

3. Установите зависимости:

```
pip install -r requirements.txt
```

4. Запустите приложение:

```
python main.py
```

После запуска откроется окно приложения, где можно выбрать аудиофайл, запустить обработку и получить транскрипт, реферирование.
